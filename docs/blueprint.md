# **App Name**: MindFlux AI Chat

## Core Features:

- AI Conversation Interface: Provides a user-friendly chat window for sending messages and receiving real-time AI responses.
- Generative AI Response Tool: Utilizes a powerful Large Language Model (LLM) to process user queries and generate coherent, contextually relevant textual replies.
- Real-time Message Streaming: Displays AI responses as they are generated, providing a dynamic and natural conversation flow.
- Backend API Proxy: A secure server-side endpoint that handles API calls to the chosen LLM service, protecting the API key from direct exposure in the frontend.
- Conversation History Display: Renders a clear, chronological view of all messages in the current conversation session.
- Responsive Input Area: A dynamic input field for user messages, including a 'send' button and basic input validation.
- Error Handling & Feedback: Provides user-friendly messages for API errors, network issues, or other problems encountered during AI interaction.

## Style Guidelines:

- A sophisticated and intelligent aesthetic is achieved with a dark color scheme. The primary color, a calming violet-blue (#8270DB), provides a vibrant yet professional feel against a very dark background. The background color (#17161D) is a subtle, deep purplish-gray derived from the primary hue, ensuring readability and focus on content. An accent color, a bright clear blue (#8CB0F2), highlights interactive elements and provides visual pop.
- The font 'Inter' (sans-serif) is recommended for both headlines and body text due to its clean, modern, and highly legible characteristics, optimizing readability for chat conversations.
- Use a set of modern, minimalistic outline icons that complement the clean interface, primarily for controls and navigation.
- The layout features a standard chat interface: a fixed input area at the bottom, and a scrollable message display area above, optimized for responsive viewing on various devices.
- Subtle and fluid animations are used for new message entries and button interactions, enhancing user feedback without being distracting.
